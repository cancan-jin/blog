# blog开发笔记
## 项目前期准备
### 安装好项目需要的开发环境
+ 开发环境
  - nodejs：后台开发语言
  - nmp：包管理工具
  - mongodb：后台数据库
  - github：代码托管
  - robo3t：mongodb可视化工具

> github安装时是选择在vs code中运行，VScode终端中输入：git config --global user.name "git用户名"，git config --global user.email "git邮箱"，提交master；在gibhub.com中新建代码仓库，返回VScode选择推送到命令，远程连接github，然后将代码上传。
